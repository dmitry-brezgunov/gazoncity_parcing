# Gazoncity Parcing
## Описание
Программа для парсинга каталога товаров сайта [ГазонCity](https://gazoncity.ru). Возвращает таблицу в xlsx c наименованием, ценой, url картинки и описанием товара.

Написана на Python с использованием библиотек BeautifulSoup для парсинга и pandas для формирования таблицы в xlsx.

## Стек
Python 3, BeautifulSoup, pandas

## Установка
На локальном компьютере должен быть установлен Python 3

1. Склонировать данный репозиторий на свой локальный компьютер.
2. Установить виртуальное окружение используя команду `python3 -m venv venv`.
3. Установить необходимые библиотеки используя команду `pip install -r requirements.txt`.
4. Запустить программу командой `python app.py`
5. После выполнения программы в корневом каталоге репозитория появится файл `gazoncity.xlsx` с собранными данными.
